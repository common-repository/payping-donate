افزونه دریافت حمایت مالی وردپرس

== Description ==
با استفاده از این افزونه میتوانید حمایت مالی کاربران را از طریق سایت وردپرسی خود و از طریق پی پینگ دریافت کنید.

== Installation ==
1. پوشه افزونه را در مسیر wp-content/plugins آپلود کنید و از حالت زیپ خارج کنید
2. افزونه را از طریق منوی \'افزونه ها\' در وردپرس فعال کنید
3. از طریق منوی حمایت مالی توکن پی پینگ خود را وارد کنید و سایر تنظیمات افزونه را تنظیم نمایید
4. در صفحه ای که میخواهید فرم دریافت حمایت مالی را قرار دهید شورت کد [PayPing_Donate] را وارد نمایید.


== Third-Party Services ==
This plugin leverages the following third-party services:

1. api.payping.ir/v3/pay/verify: To verify payments in our plugin.
2. api.payping.ir/v3/pay: For create order in our plugin and send user to gateway.
